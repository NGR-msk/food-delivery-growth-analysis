# Поиск точек роста в сервисе доставки еды: анализ пользователей и сегментация ресторанов

**Автор:** Гребенкина Наталья 
**Дата:** Февраль 2026  
**Язык:** Python (Jupyter Notebook)

---

## Описание проекта

Сервис доставки еды «Всё.из.кафе» столкнулся с типичной для рынка проблемой: рекламный трафик стабилен, установки приложения растут, но количество заказов снижается.

**Цель исследования** — найти точки роста для:
- повышения конверсии из нового пользователя в заказ;
- увеличения удержания и частоты повторных покупок;
- разработки персонализированных стратегий работы с ресторанами.

---

## Что сделано

### 1. Анализ пользовательского пути
- Построены воронки событий (классическая и step-by-step)
- Выявлен критический отток на этапе подтверждения телефона в мобильном приложении
- Конверсия confirm_phone: **Mobile 55%** vs **Desktop 90%**

### 2. RFM-сегментация клиентов
- Рассчитаны метрики: давность (R), частота (F), сумма (M)
- Выделено 5 поведенческих сегментов
- Ключевой вывод: **70% клиентов уходят после первого заказа**

### 3. Кластеризация ресторанов
- Сегментация партнёров по объёму заказов, среднему чеку и ассортименту
- Выделено 4 бизнес-модели ресторанов (от «Лидеров» до «Премиум-ниши»)

## Основные результаты

| Направление | Ключевой инсайт | Рекомендация |
|-------------|-----------------|--------------|
